# Calculadora
# Intrudução
Calculadora projetada para facilitar suas tarefas de cálculo. Seja para estudantes, profissionais ou uso cotidiano, com uma interface intuitiva e funcionalidades poderosas, estamos comprometidos em tornar suas experiências de cálculo mais rápidas, precisas e convenientes.\
Nossa calculadora foi cuidadosamente projetada para atender às suas necessidades específicas, seja para operações simples de soma e subtração ou cálculos complexos envolvendo volume, centímetros e muito mais. Queremos ser a ferramenta confiável que você pode contar em todas as situações, mas como ela ainda está em desenvolvimento, ela não terá todas as funções e pode haver bugs.\
Experimente nosso aplicativo de calculadora hoje mesmo e descubra como podemos ajudá-lo a economizar tempo, simplificar tarefas e garantir resultados precisos em todas as suas operações matemáticas, sendo abertos a reportação de bugs e/ou melhorias.
# Como Utilizar
É necessário ter o aplicativo [Visual Studio Code](https://code.visualstudio.com/download) instalado na sua máquina. Com a extensão "Live Server" para poder executa-lo.
# Funções
As funções foram separadas em 3 tipos:
- Funções Simples
- Funções de Conversor de Unidade
- Funções Matemáticas
## Funções Simples
A calculadora pode executar operações simples, podendo somar, subtrair, multiplicar e dividir. A seguir um pequeno tutorial de como executar cada função.
### Somar
Para utilizar a operação de somar, é necessário fornecer 2 (dois) números para executa-la. Tendo conhecimento dos números, faça os passos a seguir:
1. Aperte os botões correspondentes ao primeiro número que está na interface do aplicativo para poder inseri-lo.
2. Após a inserção do primeiro número, aperte o botão com símbolo "+" para seleciolar a operação de somar.
3. Com a operação somar selecionada, interir o segundo número da mesma forma que fez no 1° passo.
4. Com os dois números e a operação selecionados, aperte o botão com o símbolo "=" para obter o seu resultado desejado.

Os passos a seguir são iguais aos da função somar.

## Funções de Conversor de Unidade
A calculadora também pode executar operações de conversão de unidade, por agora, pode converter volume de galão para litro e comprimento de centimetro para metro. A seguir um pequeno tutorial de como executar cada função.
### Volume de Galão para Litro
Para utilizar a operação de conversão de unidade de volume de galão para litro, é necessário fornecer 1 (um) número para executa-la. Tendo conhecimento do número, faça os passos a seguir:
1. Acima dos botões dos números, tem um botão com o símbolo de uma régua, ao apertar nele, irá aparecer um menu de opções de Conversor de Unidade. 
2. Selecione a opção "Volume".
3. Após abrir a interface de Conversor de Unidade, você tem duas opções de execução que independendo de qual for, terá o seu resultado das duas formas.
   1. Selecione as unidades desejadas, sendo a de cima a unidade de seu número, e a de baixo a unidade que deseja converter. Em seguida inserindo o número na primeira unidade.
   2. Inserir o número na primeiro unidade. A seguir, selecionar as unidades de conversão.
### Comprimento de Centimetro para Metro
Para utilizar a operação de conversão de unidade de comprimento de centimetro para metro, é necessário fornecer 1 (um) número como na conversão de volumede galão para litro. Tendo conhecimento do número, faça os passos da função de conversão de volume, menos o 2, veja o porquê a seguir:

2. Selecione a opção "Comprimento".
## Funções Matemáticas
A calculadora também pode executar operações matemáticas, por agora, pode achar a área de um triângulo e a área de um quadrado. A seguir um pequeno tutorial de como executar cada função.
### Área do Triângulo
Para utilizar a operação de geometria plana de área do triangulo, é necessário fornecer 2 (dois) números para executa-la. Tendo conhecimento dos números, faça os passos a seguir:
1. Acima dos botões dos números, tem um botão com o símbolo de uma fórmula, ao apertar nele, irá aparecer um menu de opções de Fórmula de Matemática.
2. Selecione a opção "Geometria Plana".
3. Após abrir a interface de Geometria Plana, selecione no menu de opções do lado esquerdo qual fórmula deseja usar, selecionando a "Área do Triângulo".
4. Por fim, inserir os valores nos lugares corretos, com o resultado sendo dado automaticamente. Sendo eles:
   - b: Base
   - h: Altura
